# 特徴抽出器における分類器：ニューラル分類層とランダムフォレスト

書籍の42ページ「2.3.1 特徴抽出器としてのTransformer」に登場する、抽出された隠れ状態（ベクトル）を入力として動く2つの主要な分類アルゴリズム **「ニューラル分類層（ロジスティック回帰）」** と **「ランダムフォレスト」** について、その仕組みと違いを解説します。

---

## 1. 全体像：特徴抽出ベースの分類

このアプローチでは、Transformerモデル自体の重み（パラメータ）は完全に固定（凍結）し、単に「テキストを高品質なベクトル（数値のリスト）に変換する機械」として使用します。

```
 [ テキスト ] ──> (Transformer: 重み固定) ──> [ 隠れ状態 (ベクトル) ]
                                                      │
                                                      ▼ 特徴量として入力
                                            [ 分類器 (分類ヘッド) ] ──> [ 予測ラベル ]
                                              ★ここだけを学習する
```

この「分類器」として用いられる代表格が、ニューラル分類層とランダムフォレストです。

---

## 2. ニューラル分類層 (Neural Classification Layer)

ニューラルネットワークの最外部にくっつける、最も基本的な出力層（全結合層＋Softmaxなど）のことです。

### 最もシンプルなニューラル分類層 ＝ ロジスティック回帰
機械学習で最も定番の線形分類モデルである **「ロジスティック回帰（Logistic Regression）」** は、数学的には**「1層の全結合層（`nn.Linear`）の後にシグモイド関数やソフトマックス関数をかけた単純なニューラルネットワーク」** と全く同じ構造をしています。

* **仕組み**: 
  1. 入力された特徴量ベクトルに対し、重み（Weights）を掛け算し、バイアス（Bias）を足し算してスコア（Logits）を計算します。
  2. スコアを `Softmax` などの関数に入力し、「クラス0の確率: 15%、クラス1の確率: 85%」のように確率値に変換します。
  3. 確率が最も高いクラスを予測結果とします。
* **学習方法**: 
  * **勾配降下法（Gradient Descent）**：予測の誤り（損失）を計算し、その誤差を逆伝播させて重みを少しずつ修正（微調整）します。
* **特徴**: 
  * データが「直線（または超平面）」で綺麗にスパッと区切れる場合（線形分離可能な場合）、非常に高速で高い精度を出します。Transformerが抽出するベクトル表現は非常に優れており、線形分類しやすい状態になっているため、この単純な分類層だけで極めて高い精度が出ます。

---

## 3. ランダムフォレスト (Random Forest)

ニューラルネットワークとは全く異なるアプローチをとる、古典的な機械学習の強力な王道アルゴリズムです。

* **仕組み**: 
  1. **決定木（Decision Tree）** という、「特徴量Aが0.5以上なら右、未満なら左」といった条件分岐で予測するモデルを、ランダムにデータを選び直しながら**何百個も（森のようにたくさん）**作ります。
  2. 新しいデータを分類する際は、すべての決定木にデータを入力し、それぞれが「クラス0」か「クラス1」かを予測します。
  3. 各決定木の予測結果を集計し、**「多数決（アンサンブル学習）」** で最終的な予測クラスを決定します。

### 💡 なぜ「勾配に依存しない手法」と呼ばれるのか？
ニューラルネットワークは、数式を「微分」して「損失関数の傾き（勾配）」を求め、その傾きを頼りにパラメータの数値を少しずつずらして学習します（勾配降下法）。
これに対し、決定木（ランダムフォレスト）は**「データを最も綺麗に2グループに分けられる境界線（しきい値）はどこか？」をジニ不純度などの情報利得の指標を用いて探索し、木を二分していくアルゴリズム**です。
数式の微分や勾配計算（Backpropagation）を一切使わないため、**「勾配に依存しない（No Gradient）」** 手法と呼ばれます。

* **特徴**:
  * 直線では区切れないような、複雑にねじれた非線形なデータ境界でもうまく分類できます。
  * チューニングしなくても安定して高い精度を出してくれます。

---

## 4. 2つの手法の比較

| 比較項目 | ニューラル分類層（ロジスティック回帰） | ランダムフォレスト (Random Forest) |
| :--- | :--- | :--- |
| **学習方法** | 勾配降下法 (Gradient-based) | 木の分岐・多数決 (Gradient-free) |
| **データの境界線** | 直線 / 平面（線形） | 階段状 / 複雑な境界（非線形） |
| **計算負荷** | 🚀 非常に軽量 | 🐢 データ次元が極めて大きいと木が深く肥大化する |
| **Transformerとの相性** | 🚀 **極めて良い** (ベクトルが元々綺麗に整理されているため、線形層だけで十分高い精度が出る) | ◯ 良いが、高次元ベクトル（768次元など）に対してはやや過学習しやすい |

実務における「特徴抽出ベース」のテキスト分類では、まずは最も軽量で計算効率が良い **ロジスティック回帰（または単純な線形全結合層）** を分類ヘッドとして使うのが第一選択肢になります。
