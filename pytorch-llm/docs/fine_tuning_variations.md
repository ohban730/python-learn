# ファインチューニング (Fine-tuning) の多様な意味と定義の違い

書籍ごとの説明の違い（「モデル全体を学習する」 vs 「少量の学習で済む」）に直面した際の、**ファインチューニングという用語の広義・狭義の定義**と、パラメータ更新範囲の違いを整理して解説します。

---

## 1. 結論：ファインチューニングの定義は比較対象・文脈によって異なる

「ファインチューニング」という用語は、**比較対象（文脈）や目的によって、意味する範囲（パラメータの更新範囲やデータ量）が大きく異なります。**

* **「少量の学習で済む」という文脈**:
  **ゼロからの「事前学習 (Pre-training)」との比較**です。何千億円・数テラバイトの巨大データで学習する事前学習に対し、ファインチューニングは数千件〜数万件の**「少量のデータと短い時間」** で微調整（Fine-tune）できることを意味します。
* **「モデル全体を学習する」という文脈**:
  **前節の「特徴抽出 (Feature Extraction)」との比較**です。特徴抽出がモデルを 100% 凍結するのに対し、ファインチューニングはモデル内部の重みまで踏み込んで更新（フルファインチューニング）することを指しています。

---

## 2. パラメータ更新範囲による 3 つの分類

ファインチューニングは、**「モデルのどの層（どれくらいの割合のパラメータ）を更新するか」** によって大きく3つに分類されます。

```text
 ① フル・ファインチューニング (Full Fine-Tuning) ── 【オライリー本図2-6のパターン】
   [ 分類ヘッド ] ──> 学習する (更新)
   [ Transformer全層 (100%) ] ──> 学習する (更新)
   * 精度が最も高くなりやすいが、全パラメータを更新するため計算コスト（GPUメモリ）が大きい。

 ② 部分的ファインチューニング (Layer-wise / Partial Fine-Tuning)
   [ 分類ヘッド ] ──> 学習する (更新)
   [ Transformerの上位1〜2層 ] ──> 学習する (更新)
   [ Transformerの下位層 ] ──> ★ 凍結する (固定)
   * 言語の基礎を担う下位層は固定し、応用的な上位層だけを微調整する手法。

 ③ 効率的ファインチューニング (PEFT / 例: LoRA) ── 【現代の巨大LLMの主流】
   [ 分類ヘッド / 追加のアダプター (0.1%) ] ──> ★ ここだけ極小学習！
   [ Transformer本体 (99.9%) ] ──> ★ 100% 完全に凍結 (固定)
   * 本体モデルは1バイトも変更せず、横に付けた極小の重みだけを学習させる超省エネ手法。
```

---

## 3. なぜ「少量の学習で済む」と言われるのか？（事前学習との対比）

もう一つの視点は **「学習コスト（データ量と時間）」** です。

| 開発フェーズ | 目的 | データ量 | 学習時間 | 計算コスト |
| :--- | :--- | :--- | :--- | :--- |
| **事前学習 (Pre-training)** | 言語の文法や世界知識をゼロから覚える | 数テラバイト (数兆トークン) | 数週間〜数ヶ月 | 巨大 (数億円〜) |
| **ファインチューニング (Fine-tuning)** | 特定タスク（感情分類や指示追従）に特化させる | **数千〜数万件** | **数分〜数時間** | **少〜中 (数千円〜)** |

ほかの本（『つくって学ぶ！LLM自作入門』など）で「少量の学習で済む」と表現されているのは、この **「ゼロからの事前学習に比べて、圧倒的に少ないデータと時間（少量の学習）で自分専用のモデルが作れる」** という意味です。

---

## 4. まとめ：本ごとの表現の違いの整理

* **オライリー本（『Transformers』2章）での表現**:
  * 「特徴抽出（モデル全凍結）」との対比として、**「DistilBERT モデル全体も一緒に更新するアプローチ」** のことを指してファインチューニング（フルファインチューニング）と呼んでいます。
* **一般的なLLM解説本での表現**:
  * 「事前学習」との対比として、**「既存の学習済みモデルを少量のデータで目的のタスクに適用させるプロセス全般」** を指してファインチューニングと呼んでいます。

どちらも正しく、**「全体を更新するフル・ファインチューニングもあれば、一部やアダプターだけを更新する省エネなファインチューニングもある」** と理解しておくと、すべての本の記述が繋がります。
